## Descripción 📑

Este proyecto tiene como finalidad crear un sitio web para una productora musical ficticia que realiza ventas de licencias de sus beats, la nombré Beats Vault Studio. El foco principal estará en el desarrollo mediante HTML, CSS y el  framework bootstrap. Adicional a eso pienso desarrollar algunas funcionalidades de compra con JavaScript.
